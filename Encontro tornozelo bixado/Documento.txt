-XML
- O formato XML é uma linguagem de marcação que serve para padronizar uma sequência de dados com 
o objetivo de organizar, separar e integrar o conteúdo com outras linguagens;
- Suas principais características são sua sintaxe básica e sua portabilidade.

-JSON
- É um tipo de notação de objetos JavaScript de modo que possam ser representados de uma forma 
comum a diversas linguagens;
- Deve conter apenas informações que possam ser representadas em formato de texto;
- Os dados são separados por vírgulas;
- As chaves contém objetos;
- Os cochetes expressam matrizes/vetores.
